FoundationPress Generator creates custom-named FoundationPress themes.

== Description ==

This plugin generates custom-named versions of the FoundationPress theme. Its shamelessly copied from the fine folks at Automattic and their Components project. http://components.underscores.me

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `foundationpress-generator.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('render_types_form'); ?>` in your templates
